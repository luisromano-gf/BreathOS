# BreathOS

> *"I improve because improving is what happens when time passes."*

**BreathOS** is a living operating system for a mind navigating exponential change — specifically, the exponential expansion of applied AI.

It is not a knowledge base. Not a wiki. Not a document collection.

It is an organism. It has a compass (identity), a map (current state), an engine (adaptation process), a log (memory), and prompts (generative queries). It breathes through use. It evolves through honesty.

---

## Structure

```
/
├── compass/     → Who we are. Principles that outlast specific facts.
├── map/         → Where we are. Datestamped snapshots of the AI landscape.
├── engine/      → How we adapt. Scripts that probe, evaluate, and update.
├── log/         → How we got here. Decision journal and delta records.
└── prompts/     → How we ask. Reusable query templates for AI-assisted research.
```

## Design principles

- **Datestamp everything.** A fact without a date is a rumor.
- **Version everything.** Every past state is data about how the system evolved.
- **Markdown over proprietary formats.** Must be readable in 30 years.
- **Small files over large files.** Complexity lives in relationships, not documents.
- **The human navigates. The AI instruments.** This is a feature, not a limitation.
- **Living over finished.** v1.0 is earned through use, not designed in advance.
